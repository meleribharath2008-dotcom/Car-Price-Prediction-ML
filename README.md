# Car Price Prediction System

## Overview
This project predicts car selling prices using Machine Learning regression models.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Models Implemented
- Linear Regression
- Lasso Regression

## Workflow
1. Data Preprocessing
2. Feature Selection
3. Train-Test Split
4. Model Training
5. Performance Evaluation using R² Score
